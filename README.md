Math IA Data Analysis Quarter 1
This Power BI report provides an analysis of math assessment data for the first quarter to identify top performers and areas for improvement.

Introduction
The following analysis is a reflection of my own journey of becoming a Data Analyst as I transition from working in education. Below I will outline what you will be observing, why it is important, and how it can help teachers be more proactive by simply collecting data from their students. Enjoy! Also, feel free to reach out and give me any feedback on how to make an analysis like this better! Thank you.

Content Overview
The Power BI report includes the following sections:

<ol>
  <li><strong>Introduction and Motivation</strong>: Explanation of the purpose of the analysis and its importance.</li>
  <li><strong>Data Overview</strong>: Summary of the data used in the analysis.</li>
  <li><strong>Data Cleaning and Preparation</strong>: Steps taken to clean and prepare the data for analysis.</li>
  <li><strong>Analysis</strong>: Detailed analysis to identify trends and insights.</li>
  <li><strong>Visualization</strong>: Visualization of the findings using various charts and graphs.</li>
  <li><strong>Conclusion</strong>: Summary of findings and recommendations for teachers.</li>
</ol>
Note: All the data and values do not reflect actual performance from these schools.
